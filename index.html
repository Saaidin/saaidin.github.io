I have HTML/CSS/JavaScript skills. HTML is the most used markup language across the web, and web developers use it to create web pages on the Internet. CSS, on the other hand, is used to style the website. It's responsible for the choice of fonts, colors and layout in a website's design. Whereas HTML lays the foundation for a webpage, CSS styles it. JavaScript is a high-level programming language that implements complex features onto a website. This involves things like moving elements, search bars, videos or interactive maps on a webpage. And also, I have Responsive design skills. Because many people view webpages both online and on their mobile devices, the websites you create need to adapt to the changing screen sizes. If your website isn't responsive, this means it won't be compatible with either the desktop site or the mobile site.
